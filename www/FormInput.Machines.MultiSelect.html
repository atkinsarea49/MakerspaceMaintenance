Namespace: MultiSelect

FormInput.Machines.MultiSelect

Machine functions that allow the user to input multiple selections for one or more questions as opposed to just one. This means that the multiselect questions need to be formatted and split up so that each selection can have its own entry. See the Multiselect Machine Functions Tutorial for a breakdown of how multiselect machine functions work.
